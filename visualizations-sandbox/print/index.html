Introduction

IntroductionPrint report
This document is a sandbox for playing with visualizations within the Ovide editor.
Ovide is using vegalite peritext contextualizerhttps://github.com/peritext/peritext-contextualizer-vegalite for matters of visualization. Vegahttp://vega.github.io is a declarative format for creating, saving, and sharing visualization designs. With Vega, visualizations are described in JSON, and generate interactive views using either HTML5 Canvas or SVG.
Ovide allows to directly write visualizations specifications inside the editor - as a kind of code notebook - in order to visualize a previously imported table material. Therefore, the software can be used as a fieldbook for the documentation of a dataset exploration process.
All examples displayed in this sandbox are taken from the vegalite example webpagehttps://vega.github.io/vega-lite/examples/: you can go there to learn more about vega and its capabilities.
